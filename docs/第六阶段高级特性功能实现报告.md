### ? 1. 完整的邮件系统数据库集成

**实现位置**: `GameOfMir/M2Engine/ObjNpc.pas`

**新增脚本命令**:
```pascal
// 发送邮件到数据库

// 从数据库读取邮件
sSC_READMAILDB = 'READMAILDB';
nSC_READMAILDB = 260;

// 从数据库删除邮件
sSC_DELETEMAILDB = 'DELETEMAILDB';
nSC_DELETEMAILDB = 261;

// 从数据库获取邮件列表
sSC_GETMAILLISTDB = 'GETMAILLISTDB';
nSC_GETMAILLISTDB = 262;

// 获取邮件数量
sSC_MAILCOUNT = 'MAILCOUNT';
nSC_MAILCOUNT = 263;
```

**核心方法**:
- `ActionOfSendMailDB()` - 发送邮件到数据库
- `ActionOfReadMailDB()` - 从数据库读取邮件
- `ActionOfDeleteMailDB()` - 从数据库删除邮件
- `ActionOfGetMailListDB()` - 从数据库获取邮件列表
- `ActionOfMailCount()` - 获取邮件数量

**功能特性**:
- 支持完整的邮件CRUD操作
- 支持邮件附件功能
- 支持分页查询邮件列表
- 支持邮件数量统计
- 完整的数据库事务处理

**命令用法示例**:
```pascal
// 发送邮件到数据库
SendMailDB 收件人 主题 内容 附件

// 从数据库读取邮件
ReadMailDB 邮件ID

// 从数据库删除邮件
DeleteMailDB 邮件ID

// 从数据库获取邮件列表（第1页，每页10条）
GetMailListDB 1 10

// 获取邮件数量
MailCount
```

**技术实现**:
- 使用数据库存储邮件数据
- 支持邮件ID自动生成
- 完整的参数验证
- 数据库事务安全处理

### ? 2. 离线挂机收益计算

#### 2.1 离线收益管理
**实现位置**: `GameOfMir/M2Engine/ObjPlay.pas` 和 `GameOfMir/M2Engine/ObjNpc.pas`

**新增字段**:
```pascal
// 离线挂机收益计算
m_nOfflineExpRate: Integer;                 // 离线经验倍率
m_nOfflineGoldRate: Integer;                // 离线金币倍率
m_nOfflineItemRate: Integer;                // 离线物品倍率
m_nOfflineExpReward: Integer;               // 离线经验收益
m_nOfflineGoldReward: Integer;              // 离线金币收益
m_sOfflineItemReward: string;               // 离线物品收益
```

**新增脚本命令**:
```pascal
// 计算离线挂机收益
sSC_CALCOFFLINEREWARD = 'CALCOFFLINEREWARD';
nSC_CALCOFFLINEREWARD = 264;

// 领取离线挂机收益
sSC_CLAIMOFFLINEREWARD = 'CLAIMOFFLINEREWARD';
nSC_CLAIMOFFLINEREWARD = 265;

// 获取离线挂机收益信息
sSC_GETOFFLINEREWARD = 'GETOFFLINEREWARD';
nSC_GETOFFLINEREWARD = 266;

// 设置离线挂机倍率
sSC_SETOFFLINERATE = 'SETOFFLINERATE';
nSC_SETOFFLINERATE = 267;
```

**核心方法**:
- `ActionOfCalcOfflineReward()` - 计算离线挂机收益
- `ActionOfClaimOfflineReward()` - 领取离线挂机收益
- `ActionOfGetOfflineReward()` - 获取离线挂机收益信息
- `ActionOfSetOfflineRate()` - 设置离线挂机倍率

**功能特性**:
- 支持经验、金币、物品三种收益类型
- 支持自定义收益倍率
- 支持离线时间计算
- 支持收益领取和查询
- 完整的收益计算算法

**命令用法示例**:
```pascal
// 计算离线挂机收益（离线时间120分钟）
CalcOfflineReward 120

// 领取离线挂机收益
ClaimOfflineReward

// 获取离线挂机收益信息
GetOfflineReward

// 设置离线挂机倍率（经验150%，金币120%，物品100%）
SetOfflineRate 150 120 100
```

**技术实现**:
- 基于离线时间的收益计算
- 支持多种倍率设置
- 完整的收益分配算法
- 安全的收益领取机制

#### 2.2 离线收益计算算法
- **时间计算**: 基于玩家离线时间计算收益
- **倍率应用**: 根据设置的倍率调整收益
- **收益分配**: 自动分配经验、金币和物品
- **安全机制**: 防止重复领取和异常操作
