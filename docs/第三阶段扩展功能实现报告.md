## 实现功能清单

### ? 1. 新怪物攻击模式

#### 1.1 怪物攻击模式管理
**实现位置**: `GameOfMir/M2Engine/ObjNpc.pas`

**新增脚本命令**:
```pascal
// 设置怪物攻击模式
sSC_SETMONSTERATTACKMODE = 'SETMONSTERATTACKMODE';
nSC_SETMONSTERATTACKMODE = 229;

// 获取怪物攻击模式
sSC_GETMONSTERATTACKMODE = 'GETMONSTERATTACKMODE';
nSC_GETMONSTERATTACKMODE = 230;

// 设置怪物攻击范围
sSC_SETMONSTERATTACKRANGE = 'SETMONSTERATTACKRANGE';
nSC_SETMONSTERATTACKRANGE = 231;

// 获取怪物攻击范围
sSC_GETMONSTERATTACKRANGE = 'GETMONSTERATTACKRANGE';
nSC_GETMONSTERATTACKRANGE = 232;
```

**核心方法**:
- `ActionOfSetMonsterAttackMode()` - 设置怪物攻击模式
- `ActionOfGetMonsterAttackMode()` - 获取怪物攻击模式
- `ActionOfSetMonsterAttackRange()` - 设置怪物攻击范围
- `ActionOfGetMonsterAttackRange()` - 获取怪物攻击范围

**功能特性**:
- 支持0-9种攻击模式设置
- 支持1-10格攻击范围设置
- 通过怪物名称精确定位
- 实时动态调整怪物行为

**命令用法示例**:
```pascal
// 设置怪物攻击模式为5
SetMonsterAttackMode 怪物名称 5

// 获取怪物攻击模式
GetMonsterAttackMode 怪物名称

// 设置怪物攻击范围为3格
SetMonsterAttackRange 怪物名称 3

// 获取怪物攻击范围
GetMonsterAttackRange 怪物名称
```

**技术实现**:
- 使用`m_PEnvir.GetBaseObject()`查找指定怪物
- 直接修改怪物的`m_nAttackMode`和`m_nAttackRange`属性
- 完整的参数验证和错误处理

### ? 2. 坐骑系统升级

#### 2.1 坐骑等级和技能系统
**实现位置**: `GameOfMir/M2Engine/ObjPlay.pas` 和 `GameOfMir/M2Engine/ObjNpc.pas`

**新增字段**:
```pascal
// 坐骑系统升级
m_nHorseLevel: Integer;                     // 坐骑等级
m_nHorseSkills: array[0..9] of Integer;     // 坐骑技能等级数组
```

**新增脚本命令**:
```pascal
// 升级坐骑
sSC_UPGRADEHORSE = 'UPGRADEHORSE';
nSC_UPGRADEHORSE = 233;

// 获取坐骑等级
sSC_GETHORSELEVEL = 'GETHORSELEVEL';
nSC_GETHORSELEVEL = 234;

// 设置坐骑技能
sSC_SETHORSESKILL = 'SETHORSESKILL';
nSC_SETHORSESKILL = 235;

// 获取坐骑技能
sSC_GETHORSESKILL = 'GETHORSESKILL';
nSC_GETHORSESKILL = 236;
```

**核心方法**:
- `ActionOfUpgradeHorse()` - 升级坐骑等级
- `ActionOfGetHorseLevel()` - 获取坐骑等级
- `ActionOfSetHorseSkill()` - 设置坐骑技能
- `ActionOfGetHorseSkill()` - 获取坐骑技能

**功能特性**:
- 支持坐骑等级1-100级
- 支持10个坐骑技能槽位
- 每个技能支持0-3级
- 完整的坐骑检查机制

**命令用法示例**:
```pascal
// 升级坐骑5级
UpgradeHorse 5

// 获取坐骑等级
GetHorseLevel

// 设置坐骑技能ID为3，等级为2
SetHorseSkill 3 2

// 获取坐骑技能ID为3的等级
GetHorseSkill 3
```

**技术实现**:
- 使用`m_UseItems[U_House].wIndex`检查坐骑存在
- 等级上限100级保护
- 技能数组索引0-9，等级0-3
- 完整的参数验证和用户提示
