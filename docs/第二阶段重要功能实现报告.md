## 实现功能清单

### ? 1. 装备品质和特效系统

#### 1.1 装备品质系统（0-4阶）
**实现位置**: `GameOfMir/M2Engine/ObjNpc.pas`

**新增脚本命令**:
```pascal
// 设置装备品质
sSC_SETITEMQUALITY = 'SETITEMQUALITY';
nSC_SETITEMQUALITY = 215;

// 获取装备品质
sSC_GETITEMQUALITY = 'GETITEMQUALITY';
nSC_GETITEMQUALITY = 216;
```

**核心方法**:
- `ActionOfSetItemQuality()` - 设置装备品质等级
- `ActionOfGetItemQuality()` - 获取装备品质等级

**技术实现**:
- 使用`temp1[1]`字段存储装备品质等级（0-4阶）
- 支持通过MakeIndex指定装备
- 完整的参数验证和错误处理

**命令用法示例**:
```pascal
// 设置装备品质为3阶
SetItemQuality 装备编号 3

// 获取装备品质
GetItemQuality 装备编号
```

### ? 3. 离线挂机系统

#### 3.1 离线挂机管理
**实现位置**: `GameOfMir/M2Engine/ObjPlay.pas` 和 `GameOfMir/M2Engine/ObjNpc.pas`

**新增字段**:
```pascal
// 离线挂机系统

m_dwOfflineStartTime: LongWord;             // 离线挂机开始时间
m_dwOfflineDuration: LongWord;              // 离线挂机持续时间
m_sOfflineMapName: string;                  // 离线挂机地图名称
```

**新增脚本命令**:
```pascal

nSC_STARTOFFLINE = 223;

// 停止离线挂机
sSC_STOPOFFLINE = 'STOPOFFLINE';
nSC_STOPOFFLINE = 224;

// 检查离线挂机状态

nSC_CHECKOFFLINE = 225;
```

**核心方法**:
- `ActionOfStartOffline()` - 开始离线挂机
- `ActionOfStopOffline()` - 停止离线挂机
- `ActionOfCheckOffline()` - 检查离线挂机状态

**功能特性**:
- 支持指定挂机时间（分钟）
- 支持指定挂机地图
- 实时状态检查和剩余时间显示
- 完整的挂机管理功能

**命令用法示例**:
```pascal

// 停止离线挂机
StopOffline

// 检查离线挂机状态
CheckOffline
```

**技术实现**:
- 使用毫秒级时间戳管理挂机时间
- 支持地图名称指定
- 完整的状态管理和用户提示
